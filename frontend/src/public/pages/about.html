About the project

A simple e-commerce application built to practice core JavaScript concepts.

Go home

This project is a small e-commerce application built using vanilla JavaScript with no front-end frameworks. The goal of this project is to practice and maintain my existing JavaScript skills. Everything used here is already familiar, and the focus is on not losing touch with core JavaScript fundamentals.

The UI is styled using Tailwind CSS and DaisyUI, while routing and API handling are implemented with Express.js.

Key Features

  • Product listing with category filtering
  • Dynamic product detail pages
  • Shopping cart with total items and price calculation
  • Checkout page summary
  • Client-side routing handled via Express
  • Responsive layout for mobile and desktop
  • Unit and integration tests

Tech Stack

Vanilla JavaScript Express.js Tailwind CSS DaisyUI HTML CSS Testing

This project is intended for learning and practice purposes only.
